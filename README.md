# Secure Password Vault 
 
A CLI-based encrypted password manager built in Python using Fernet encryption and PBKDF2 key derivation. 
 
## Features 
- Master password authentication 
- PBKDF2-HMAC-SHA256 key derivation 
- Fernet symmetric encryption 
- Secure salt storage 
- Encrypted credential storage 
 
## Tech 
- Python 
- cryptography
